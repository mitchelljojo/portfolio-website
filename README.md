# Jojo Portfolio

CV + portfolio website for Jojo, focused on analytics engineering, data analysis, and selected project work.

## Run Locally

Prerequisite: Node.js 18 or newer.

```bash
npm install
npm run dev
```

Then open the local URL shown in your terminal. The dev script is configured for:

```text
http://127.0.0.1:3000
```

## Useful Commands

```bash
npm run lint
npm run build
npm run preview
```

## Edit Content

Portfolio text, projects, skills, experience, education, and certifications are managed in:

```text
src/data.ts
```

The generated hero image is stored at:

```text
public/floral-data-banner.png
```
